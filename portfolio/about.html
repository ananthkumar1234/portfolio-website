Experienced in developing dynamic and interactive web applications using Java technologies.

Java Frameworks:
Familiarity with Java frameworks such as Spring and Hibernate for building scalable and efficient web applications.

Web Services:
Proficient in creating and consuming web services, including RESTful APIs.

Experience with version control systems like Git for collaborative development and code management.
